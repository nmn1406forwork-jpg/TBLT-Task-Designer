      💡 Gamification techniques to consider

      Whether or not you use the escape room format, these mechanics can add motivation to any task sequence:

      • Point systems — award points for completing each stage, using the target language correctly, or finishing within a time limit.
      • Leaderboards — display team or individual rankings to add light competitive pressure across a class or multiple groups.
      • Badges or rewards — give a small visual reward (a sticker, stamp, or digital badge) for unlocking each stage of the sequence.
      • Timers — set a soft time limit per stage to build pace and urgency without penalising accuracy too heavily.
      • Team roles — assign roles such as "clue keeper" or "writer" so every learner has a clear responsibility within the activity.
